Github

Utilsation de Github à l'aide du terminal

Commande Action
git init Permet de créer un nouveau dépot
git add Indique le suivi d'un fichier
git commit -m Création d'une sauvegarde, en vue de l'exportée sur github
git remote -u + URL Indique l'URL du repository au dépot
push -u origin master Envoie du fichier vers le repository Github
git status Indique le suivis d'un dépot
git log Hisotrique d'un dépot
git branch + nom Création d'une branche pour un repo
git checkout + nom de la branche Basculer sur une branche donnée
git checkout -d Retour sur la branche master
git branch -d +nom branche suppression d'une branche donnée
git merge + nom branche Fusionner la branche sur master
git pull Télécharger un repo git

Tuto

Dans le terminal

  1. Créer un répertoire qui servira de base de travail (mkdir)
  2. Créer un fichier (.html, .css ...) (touch)
  3. Appeler le fichier dans un editeur de texte (nom editeur de texte + fichier)

    4. (travail sur fichier)

  4. Créer un dépot (git init)
  5. Suivre un fichier (git add + nom fichier)
  6. Sauvegarde (git commit -m "commentaire")

Sur Github

  1. Créer un nouveau repo (new repository)
  2. Copier-coller l'URL donnée

Sur le terminal

  1. Lier le dépot à l'URL (git remote add origin + URL)
  2. Envoyer le fichier vers Github (git push -u origin master)

Penser à suivre les sauvegardes sur le terminal avec git status et git log.
